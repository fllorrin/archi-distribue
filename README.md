# archi-distribue
Pour le moment vous poouvez lancer keycloak comme cela :
    -aller dans le dossier infrastructure
    
    - ouvrir un terminal
    
    - faire un "docker compose up -d"
    
    - puis aller sur http://localhost:8080/admin, identifiant : admin, mdp : admin
    
    - Vous arrivez donc sur keycloak avec le realm LoCaRe
